# Aletheia
Latest Deployment link: https://aletheia-lilac.vercel.app/

To run the application, run following commands in order :

```
yarn install
yarn prisma generate
yarn dev
```

To get a lightweight GUI for viewing and editing database tables, run:

```
yarn prisma studio
```
